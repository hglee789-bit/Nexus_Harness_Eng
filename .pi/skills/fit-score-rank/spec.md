# spec.md — fit-score-rank Skill

## 목표
사용자 프로필과 공고 정보를 비교하여 6가지 항목별 점수를 계산하고, 총 적합도 점수(0–100)와 breakdown을 생성한다.

## 맥락
**대상:** 사용자 프로필 JSON, 파싱된 공고 JSON  
**사용 목적:** Fit & Priority Agent에서 호출하여 개인화 점수 계산

## 범위

**포함:**
- 전공 적합도 (0–25점): 학과 ↔ 공고 분야 매칭
- 관심 분야 적합도 (0–25점): 키워드 일치도
- 희망 직무 적합도 (0–20점): 직무 연결성
- 참가 자격 적합도 (0–15점): 학년, 자격 조건
- 일정 가능성 (0–10점): 마감까지 남은 기간
- 지역/온오프라인 (0–5점): 활동 위치 일치
- 각 항목별 이유 기록

**제외:**
- 최종 우선순위 결정 (deadline-priority-rank가 담당)

## 입력 예시

```json
{
  "user_profile": {...},
  "opportunity": {...}
}
```

## 출력 예시

```json
{
  "fit_score": 88,
  "fit_breakdown": [
    {"category": "전공 적합도", "max": 25, "earned": 20},
    {"category": "관심 분야", "max": 25, "earned": 25}
  ]
}
```

## 성공 기준

- [ ] 점수 합계가 정확히 0–100 범위다
- [ ] 각 항목별 breakdown이 포함된다
- [ ] 같은 입력에는 같은 점수가 나온다
